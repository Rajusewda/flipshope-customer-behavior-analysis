# Customer Behavior Analysis in E-commerce

This project analyzes customer behavior on an e-commerce platform using demographic, transactional, and engagement data.

## Objective
To identify key drivers of customer satisfaction and develop retention strategies based on customer segmentation and purchasing behavior.

## Dataset Features
- Customer ID
- Gender
- Age
- City
- Membership Type
- Total Spend
- Items Purchased
- Average Rating
- Discount Applied
- Days Since Last Purchase
- Satisfaction Level

## Analysis Performed
- Data cleaning and preprocessing
- Customer segmentation
- Spending analysis
- Geographic insights
- Satisfaction analysis
- Retention risk identification
- Correlation analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Premium members show higher spending behavior.
- Customers inactive for more than 60 days show churn risk.
- Discount campaigns influence purchasing behavior.
